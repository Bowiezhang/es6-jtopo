# es6-jtopo

## 版本更新记录

- 0.1.0   
  初始化项目，封装了jtopo的方法，提供node拖动时自动对齐功能

## 安装  

npm install es6-jtopo

## 实例  

git clone https://github.com/Bowiezhang/es6-jtopo.git

主目录为topo封装库，测试项目在tests目录中

```
npm link

cd tests

npm install

npm link es6-jtopo

npm run start
```

## 使用
```
import Topo from 'es6-jtopo'

new Topo()
```
